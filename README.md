# MoodleShortcuts

## Setup instructions

1. Install required packages on your computer

    ```zsh
    brew install yq
    ```

2. Set up config

    - Copy `config/config-example.yml` as `config/config.yml`
    - Edit `config/config.yml` appropriately for your circumstances.

3. Run generate.sh

    ```zsh
    chmod +x generate.sh
    ./generate.sh
    ```
